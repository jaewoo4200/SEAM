SEAM Studio

로컬 우선 AI 지원 RF 디지털 트윈 워크벤치. 하나의 텍스처 3D 씬에서 모든 메시가 시각(PBR)과 전자기(RF) 두 개의 재질 바인딩을 지니고, 캐노니컬 씬을 Sionna RT 프로젝션으로 컴파일해 레이 경로·라디오맵을 같은 뷰포트 위에 되돌립니다. GPU도, Sionna도, LLM도 필수가 아닙니다.

What it does

씬 임포트에서 전파 시뮬레이션까지, 한 화면에서

Mitsuba XML·zip 번들·OpenStreetMap 사각형 선택까지 임포트하고, 재질을 할당·검증하고, 28 GHz 레이트레이싱을 돌리고, 결과를 다시 그 씬 위에 얹습니다.

Author

씬 & 재질 저작

텍스처 임베디드 GLB 뷰어(Blender식 조작), 프림별 RF 재질, 물리 분할.

  • zip 번들 / OSM 지도 드래그 임포트
  • 재질·연결성 기반 메시 분할 + undo
  • ITU 대역 가드레일 & 검증 린트
Simulate

Sionna RT 프로젝션

캐노니컬 씬 → Mitsuba XML 자동 컴파일, 스테일 자동 재컴파일.

  • 경로·라디오맵·SINR·빔포밍·채널(CFR)
  • 다중 UE 궤적 + 지형 추종 드레이프
  • ML ground-truth 데이터셋 내보내기
SEAM-Agent

검색증강 로컬 AI 재질 할당

로컬 LLM/VLM 하니스가 건물을 멀티뷰로 관찰하고 웹 증거로 보강해 재질을 제안.

  • triangle-id 역투영 → 면 단위 세그먼트
  • 관측 가능한 activity trace + 증거 카드
  • confidence·provenance·사용자 승인

Pipeline

unsegmented mesh → Sionna-ready RF scene

1 · Import

Mitsuba XML / zip / OSM. 텍스처는 뷰어 GLB와 AI 증거용 원본으로 이중 보존.

2 · Segment

휴리스틱 · 로컬 VLM · 외부 SAM2 마스크로 면 단위 분할.

glassconcretemetalground
3 · Assign

SEAM-Agent 제안 + 사용자 리뷰. 모든 결정이 provenance에 기록.

4 · Compile

재질 그룹별 PLY + 깨끗한 ITU bsdf의 Mitsuba XML 자동 생성.

5 · Solve

Sionna RT(또는 Mock) 경로·라디오맵 → 같은 뷰포트에 오버레이.

Capabilities

기능 자세히 보기

위 세 기둥 아래에는 링크 분석부터 데이터셋 생성까지, 논문·실험에 바로 쓰는 도구들이 들어 있습니다.

채널 분석 & 라이브 튜닝

링크버짓, CIR/CFR, K-factor, RMS 지연확산, 경험 경로손실 모델(3GPP 38.901 포함) 대 RT 비교. 주파수·대역폭·TX 파워·잡음지수·SCS를 바꾸면 즉시 재분석되고, TS 38.215 스타일 RSRP/RSSI/RSRQ가 함께 산출됩니다.

다중 TX 간섭 · SINR

서빙 TX 이외 모든 TX의 레이트레이싱 수신전력을 동일채널 간섭으로 합산해 SINR = S/(I+N)을 계산합니다. 서빙 셀 선택, SINR 라디오맵, 셀별 서빙 TX 지도까지.

MIMO 빔포밍

코드북 스윕 / TX-MRT / SVD 빔포밍 이득을 실제 레이 채널 위에서 계산합니다 (4×4에서 ~12/24 dB 검증).

메시 라디오맵 + 영역 정밀화

수평 평면이 아니라 벽·바닥·도로 표면 위 삼각형 단위로 커버리지를 칠하고, 관심 영역만 더 작은 셀로 다시 풉니다.

궤적 RF 지표 & 지형 추종

UE 궤적을 따라 RSS·path gain·RMS delay·SINR을 샘플링합니다. 궤적은 지형과 지붕 표면에 드레이프되어 언덕을 뚫고 지나가지 않고, 디바이스는 지표고도(AGL)로 배치합니다.

재질 세그멘테이션 & 분할

통짜 건물 메시를 텍스처 마스크(컬러 휴리스틱 / 로컬 VLM / SAM2 마스크 업로드)로 면 단위 재질 분할하고, 병합 메시는 연결 성분으로 건물별 분리합니다. 모든 분할은 GLB 백업과 함께 undo 가능합니다.

측정 기반 캘리브레이션

실측 링크 CSV를 불러와 재질 파라미터를 그리드 피팅하고, 시각적으로 같은 재질을 측정 path gain으로 판별하며, 재질 지정이 링크에 미치는 임팩트를 NMSE·용량으로 정량화합니다.

Metrics dashboard & 논문용 내보내기

링크 KPI와 CIR·CFR·도플러·경로손실 차트를 흰 배경 serif 논문 스타일로 — 차트마다 PNG/SVG/CSV 버튼 내장. 뷰포트 캡처와 Mitsuba 오프라인 렌더도 지원합니다.

데이터셋 & AODT 연동

ML ground-truth 데이터셋(npz + metadata) 생성, AODT RFData 컨트랙트 내보내기, NVIDIA AODT parquet 결과를 같은 스키마로 가져오기.

자연어 규칙 & 검증 설명

"창문은 유리, concrete 벽은 itu_concrete" 한 문장을 검토 가능한 지정 규칙으로 바꿔 일괄 적용하고, 검증 경고를 평문으로 풀어 원클릭 조치와 함께 설명합니다.

결과 재현성 & 라이브 이벤트

모든 결과에 씬·지정·설정 해시와 config 스냅샷을 각인해 stale 결과를 배지로 알리고, WebSocket으로 컴파일·시뮬레이션 진행을 스트리밍합니다.

Live sync POST /live/state

GPS·모캡·로그의 실세계 위치를 로드된 씬에 주입하면 뷰어가 실시간으로 따라 움직이고, 즉시 재시뮬레이션해 measure → sync → predict 루프를 돌립니다.

Quickstart

세 줄이면 켜집니다

git clone https://github.com/jaewoo4200/SEAM.git && cd SEAM
powershell -ExecutionPolicy Bypass -File scripts\install.ps1   # or: bash scripts/install.sh
powershell -ExecutionPolicy Bypass -File scripts\start.ps1     # open http://localhost:5173

Python 3.11+ · Node 20+. GPU/Sionna/LLM 없이 Mock 백엔드로 전 기능 체험 가능 — Sionna RT·LM Studio는 선택적 업그레이드입니다. 자세한 내용은 INSTALL.md · TUTORIAL.md.
